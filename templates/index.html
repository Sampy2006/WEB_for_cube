наши направления

Основы программирования на Java

Задача программиста Java — написать новый код, чтобы мобильное приложение, мессенджер или веб-сайт могли работать. Также специалисты сотрудничают с дизайнерами, project-менеджерами, тестировщиками и другими техническими членами команды.

Программирование на Python

Обучение по программе Яндекс-Лицея. Программа имеет базовый уровень сложности.Python предусматривает знакомство с теоретическим материалом и практические занятия по решению задач на закрепление пройденного.

Основы алгоритмики. ПиктоМир

ПиктоМир позволяет ребенку "собрать" из пиктограмм на экране компьютера несложную программу, управляющую виртуальным исполнителем-роботом.

Мобильная разработка

Разработчик мобильных приложений (mobile-разработчик) – это программист, который конструирует, разрабатывает и модернизирует программное обеспечение для носимых и мобильных цифровых устройств.

Цифровая гигиена и работа с большими данными

Программа направлена на обучение детей и подростков ориентироваться в информационном пространстве,проверять, оценивать получаемую информацию.

Шахматы

Секция направлена на обучение играть детей и подростков в шахматы

Системное администрирование

Системное администрирование – процесс создания, настройки и проведения других техническихмероприятий, направленных на поддержание информационных систем обработки и передачи данных и разграниченного доступа к ним в рабочем состоянии.

Разработка VR/AR-приложений

Обучающиеся разрабатывают образовательные приложения, проектируют симуляторы для будущих инженеров, проводят виртуальные туры по культурным и историческим достопримечательностям и др.

Программирование на Scratch

Создавать при помощи Scratch можно интерактивные мультфильмы, компьютерные игры, крутые презентации, научные эксперименты, учебные программы, которые пригодятся в школе.

Лаборатория программирования роботов

Программирование роботов - это разработка схемы управления тем, как машина взаимодействует с окружающей средой и достигает своих целей.
